## Task Manager Computer Use

### Safety

Task Manager can end processes and change startup behavior. Do not click End task, Efficiency mode, Startup disable, or service controls unless the user explicitly requests it.

### Inspection

Use `dump_app_targets` for tabs, search, column headers, and menu buttons. Use screenshots and the visible table before reporting process names or resource usage.

### Navigation

Re-query after switching pages because the process table, details view, services view, and startup apps view expose different target sets.

### Localization

Prefer control type, AutomationId, table structure, and window-relative positions over English column labels.
